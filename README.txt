El proyecto está organizado de la siguiente forma:

Carpetas:
    - clasificadores: implementación de los clasificadores usados.
    - datos: los datos para el ejemplo de digidata
    - pruebas: resultados e imagenes de las pruebas hechas y que complementan a las diapositivas

Archivos:
    - resultadosAleatorios.py: ejemplo de como ejecutar las pruebas para los conjuntos generados aleatoriamente
    - resultadosDigi.py: ejemplo de como ejecutar las pruebas para los datos de digidata
    - resultadosVotos.py: ejemplo de como ejecutar las pruebas para los datos de Votos.py
    - presentacion.pdf: presentación de clase
    - procesamiento.py: archivo con las funciones para leer los datos de digidata y convertirlos a funciones de python
